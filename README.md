# asia-mjgames-archive-1e92

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，内容涉及亚洲地区棋牌游戏相关的资料存档。页面来源于公开渠道，仅供学习、研究和历史参考使用。

## 目录说明

```
.
├── README.md              # 本文件
├── index.html             # 入口页面（可选）
├── pages/                 # 存放归档的 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/                # 页面引用的静态资源（图片、样式、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下保存每个独立归档的 HTML 文件。
- `assets/` 目录存放页面所需的公共资源文件，按类型组织。

## 页面归档说明

- 每个 HTML 页面均为独立归档，不依赖于特定域名或网站。
- 归档内容包括页面结构、样式、脚本及文本信息，尽可能保留原始外观。
- 部分页面可能因外部资源失效而显示不完整，不影响本地阅读。
- 页面内容仅供存档用途，不代表任何立场或观点。
